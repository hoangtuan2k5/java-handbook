# Evergreen Java Notes

## Purpose

Folder này dành cho principle, heuristic, design trade-off, và mental shortcut có giá trị lâu dài trong nhiều context Java/Spring khác nhau.

Mục tiêu là lưu ý tưởng bền vững, không phải nhật ký học ngắn hạn.

## What belongs here

- Một principle có thể đọc lại sau vài tháng vẫn hữu ích.
- Một heuristic giúp ra quyết định giữa nhiều lựa chọn đúng.
- Một trade-off xuất hiện lặp lại trong Java/Spring production code.
- Một mental shortcut đã được kiểm chứng qua nhiều note hoặc bug lesson.

## What does not belong here

- Câu hỏi chưa chắc chắn; để ở `05_Open-Questions/`.
- Bug cụ thể; để ở `04_Lessons-from-bugs/`.
- Tutorial API chi tiết; để ở `01_Core/` hoặc `02_Advanced/`.

## Suggested structure

```markdown
# Evergreen idea

## Core idea

## Why it matters

## Example

## When this breaks down

## Related notes
```
