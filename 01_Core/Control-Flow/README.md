# Control Flow

## Purpose

Folder này ghi lại các cấu trúc điều khiển luồng cơ bản trong Java: loop, switch, recursion, và cách thoát khỏi block/method.

Mục tiêu là giúp code đọc tự nhiên, tránh loop vô hạn, recursion quá sâu, hoặc dùng `break`/`continue`/`return` làm flow khó theo dõi.

## Suggested reading order

1. `001-for-vs-while-vs-do-while.md`
2. `002-switch-expression.md`
3. `003-break-vs-continue-vs-return.md`
4. `004-recursion.md`

## How to use this folder

- Khi chọn loop, bắt đầu từ `001`.
- Khi code có nhiều nhánh điều kiện, xem `002` để cân nhắc switch expression.
- Khi method khó đọc vì nhiều exit point, đọc `003`.
- Khi dùng recursion, đọc `004` cùng bug lesson về `StackOverflowError`.
