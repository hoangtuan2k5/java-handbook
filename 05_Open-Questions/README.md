# Java Open Questions

## Purpose

Folder này lưu các câu hỏi còn mở, giả thuyết chưa chắc, hoặc topic cần research thêm trước khi biến thành note chính thức.

Mục tiêu là không giả vờ chắc chắn khi evidence chưa đủ.

## Current notes

- `001-volatile.md`: câu hỏi mở quanh `volatile`, visibility, atomicity, và thread-safety.

## How to use this folder

- Mỗi note phải có question rõ ràng, current hypothesis, evidence đã có, phần còn thiếu, và next step.
- Khi câu hỏi đã resolve, chuyển kết luận sang `01_Core/`, `02_Advanced/`, hoặc `03_Evergreen-Notes/` rồi link ngược lại.
- Không để note thành bãi chứa link; mỗi link phải nói nó giúp trả lời phần nào.
